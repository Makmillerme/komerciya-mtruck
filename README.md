# 🚀 Генератор PDF Комерційних Пропозицій

**Веб-додаток для створення професійних PDF комерційних пропозицій на Windows та Mac.**

## 📋 Можливості

- ✅ Веб-інтерфейс для введення даних
- ✅ Завантаження до 4 фото (JPG, PNG, WEBP, AVIF)
- ✅ Автоматична генерація PDF
- ✅ Завантаження PDF прямо з сайту
- ✅ Кроссплатформна (Windows, Mac)
- ✅ Простий та зручний інтерфейс

## 🚀 Швидкий старт

### Windows
**Просто подвійний клік на `start.bat`** - все запуститься автоматично!

### Mac

**Просто подвійний клік на `start.command`** - все запуститься автоматично!

> ⚠️ **При першому запуску macOS може попросити дозвіл:**
> 1. Клікніть правою кнопкою (або Ctrl+клік) на `start.command`
> 2. Виберіть **"Відкрити"**
> 3. Підтвердіть **"Відкрити"** в діалоговому вікні
> 4. **Готово!** Після цього подвійний клік працюватиме завжди без додаткових дій

## 📱 Використання

1. Запустіть програму (`start.bat` або `start.command`)
2. Браузер відкриється автоматично на http://localhost:5000
3. Заповніть форму з усіма даними
4. Завантажте до 4 фото
5. Натисніть "🚀 Згенерувати PDF"
6. Натисніть "📥 Завантажити PDF"

## 🔧 Troubleshooting

**Помилка: "Flask not found"**
```bash
pip install Flask
```

**Помилка: "Playwright not installed"**
```bash
playwright install chromium
```

**Додаток не відкривається**
- Перевірте, що порт 5000 не зайнятий
- Спробуйте http://127.0.0.1:5000

**PDF не створюється**
- Перевірте, чи всі поля заповнені
- Перевірте, чи фото завантажені правильно
- Перегляньте помилку в консолі
